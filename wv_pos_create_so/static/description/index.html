POS create sale order

This module helps us to create sale order from point of sale.

Features
  • Seller can easily create sale order from point of sale.
  • Customer can easily book their orders and get the receipt.
  • We can print the receipt by thermal printer or normal printer.
1. Allow sale order in POS.
2. Sale order Confirm state.
3. Create sale order button.
4. Order Reference.
5. Sale order receipt.
6. Sale order created from POS.
